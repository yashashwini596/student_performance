 Student Performance Prediction

Machine Learning project that predicts a student's final percentage based on study hours, attendance, assignments, and previous marks using **Linear Regression**.

---

 Project Overview

This project uses data analysis and machine learning to predict a student’s academic performance. It includes:

- Dataset preprocessing
- Model training using **Linear Regression**
- Accuracy evaluation
- Web-based user interface using **Streamlit** to input student data and get predictions

---

 Features

- Predict student percentage based on inputs
- Linear Regression model for prediction
- Visualize actual vs predicted results
- Easy-to-use **Streamlit** interface

---

Dataset Features

The dataset contains the following features:

| Feature | Description |
|---------|-------------|
| `study_hours` | Number of hours the student studies per day |
| `attendance` | Attendance percentage of the student |
| `assignments` | Number of assignments completed |
| `previous_marks` | Marks obtained in previous exams |
| `percentage` | Final percentage (target variable) |
